[English]
*HOW TO RUN IT FOR THE FIRST TIME*

1) Change database connection string in Web.config (using your server and catalog)

2) Enable migrations in Persistance
enable-migration

3) Delete existing migrations and run again
Add-migration Initialize

4) Update database
update-database

5) Select default execution project
WebApp (you may change its name)



[Español]
*COMO EJECUTARLO POR PRIMERA VEZ*

1) Cambiar cadena de conexion de base de datos en Web.config (el server y el catalog)

2) Habilitar migraciones en Persistance
enable-migration

3) Borrar las migraciones existentes y ejecutar de nuevo
Add-migration Initalize

4) Actualizar base de datos
update-database

5) Seleccionar proyecto de ejecución por defecto 
WebApp (puedes cambairle el nombre al proyecto)
